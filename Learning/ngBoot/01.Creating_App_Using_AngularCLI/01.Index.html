Installing the package

Run the following code:

npm install -g @angular/cli

Create a new project

To create a new skeleton application run the following command:

ng new my-app

Serve the application

Navigate to the folder, `my-app`, and run the following command, which will open the browser and launch application at http://localhost:4200/

ng serve --open

Additional Notes

To build the application

Following command can be used to build the applicaiton without serving. We can also specify the environment (optional):

ng build ng build --prod

Running Unit tests

Use the following command to run unit tests (the spec.ts tests):

ng test

Running End-to-End tests

ng e2e
